# PDF Architect Remediation script

### Actions
- Terminates processes associated with PDF Architect unwanted software
- Remove PDF Architect unwanted software from the file system.
- Remove PDF Architect unwanted software registry keys.
- Remove PDF Architect uninstall key.
- Performs checks to ensure removal was completed and lets you know if it wasn't successful.

### Description
I have created an automated script to remove PDF Architect. The script is intended for CrowdStrike but can be run without it.
